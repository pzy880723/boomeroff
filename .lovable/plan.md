## 目标

当前撰稿弹窗把「实时预览」占了屏幕最大一块，对话被压在最底部 180px 内、输入体验差；AI 引导又太啰嗦（每次都要追问 2-3 个问题）导致「写不出来」。这次重做整个 AI 写稿功能：**把对话变成主界面，预览按需查看**。

## 新交互（手机端为主）

弹窗改为 **对话为主 + 双视图（对话 / 预览）左右滑动切换**：

```text
┌───────────────────────────────┐
│ AI 撰稿           [对话|预览] │  ← 顶部分段控件（Segmented）
├───────────────────────────────┤
│                               │
│  💬 对话区（占满，可长输入）  │
│                               │
│  [快捷片：📢公告 📋制度       │
│   🎉活动 🚨紧急 · 通知/资讯] │
│                               │
├───────────────────────────────┤
│  [👁 预览] 输入框…    [发送]  │  ← 底部悬浮"预览"按钮 + 输入
└───────────────────────────────┘
```

切到「预览」Tab：显示 Banner + 标题 + 正文 + 工具栏（上传/AI 画封面/插图/编辑/发布）。

三种进入预览的方式：
1. 顶部分段控件点「预览」
2. 输入框左侧「👁 预览」按钮（有草稿时才亮，草稿生成后自动弹一个 Toast「已生成，点击查看预览 →」）
3. **左右滑动手势**（Swipe：对话 → 左滑到预览，预览 → 右滑回对话）

## AI 引导重做（解决"AI 撰稿不行"）

现在系统提示强制"先追问 2-3 个问题"，用户觉得啰嗦。改为：

1. **一句话直出模式**：默认用户说什么就直接出稿，不追问。除非信息严重不足（<8 个字且无关键词）才追问 1 个最关键的问题。
2. **快捷模板**：对话区上方常驻 4 个模板 chip：`📢 发通知` `📋 发制度` `🎉 发活动` `🚨 紧急通知`；点击后自动填一句 prompt 提示（"发一条…活动通知，主题是___"）并 focus 输入框。
3. **流式渲染**：`spirit-chat` 已支持 SSE，改用流式 append，用户能看到字在打，等待感消失。
4. **稿件生成后**：AI 消息下面直接显示两个内联按钮 `[查看预览]` `[让我改一下：更短/更正式/加数据]`，一键调整。
5. **多次生成保留历史**：每次 AI 出稿都把 `{title, body}` 存到"版本"列表，预览页顶部可下拉切换历史版本（防止改坏了要重来）。
6. **JSON 解析容错**：现有 `/\{[\s\S]*\}/` 会误吞正文里的花括号；改用带围栏识别（```json … ``` 优先），失败时仅提取带 `"title"` 和 `"body"` 键的对象。

## 技术改动清单

**只改 `src/pages/Notifications.tsx`**，其它文件不动：

- 用 `useState<'chat'|'preview'>('chat')` 管理视图；`Tabs` 组件承载切换。
- 用 `onTouchStart/onTouchEnd` 或复用现有 `useSwipe`（若无则内联 60px 阈值判定）实现左右滑动。
- 底部输入区左侧加 `Eye` 图标按钮，`title||body` 存在时高亮 + 显示小红点。
- 对话消息渲染增加 `parts`：`{type:'text'}` 与 `{type:'draft-actions'}`（内联按钮）。
- 生成草稿后：`toast.success('草稿已生成', { action:{ label:'查看', onClick:()=>setView('preview') } })`。
- system prompt 重写为「默认直出、极少追问、支持迭代改写」。
- 新增 `versions: {title,body,at}[]` 状态与预览页顶部 `Select` 版本切换。

## 保留不变

- 分类 / 类型 / Banner 裁剪 / AI 生成 Banner / 插入正文图 / Markdown 渲染 / 发布逻辑全部保留，只是搬到「预览」Tab 内。
- 后端 `spirit-chat` 和 `generate-notification-banner` edge functions 不改。
- 底栏红点、消息中心其它 Tab 都不动。

## 验收

- 打开撰稿：默认停在「对话」Tab，输入框可见、可长输入。
- 简单说一句「明天早会 9 点」→ AI 直接出稿（不再连问 3 个问题）。
- 出稿后：Toast 提示 + 输入框左侧「预览」按钮变亮 + 消息内联「查看预览」按钮。
- 从对话向左滑 → 平滑切到预览页；预览页向右滑 → 回对话。
- 预览页顶部可切换历史版本；工具栏一切照旧；发布正常。
