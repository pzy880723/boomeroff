# Surprise Pro 1080p Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the surprise video flow submit Seedance 2.0 Pro at 1080p end to end while preserving explicit 720p recovery.

**Architecture:** Keep the employee flow fixed and simple in the React dialog, add server-side defaults in the surprise Edge Function for old clients, and centralize model/resolution normalization in the existing Seedance model helper. The generic renderer consumes that helper so omitted resolution follows the selected model rather than a global 720p fallback.

**Tech Stack:** React, TypeScript, Supabase Edge Functions, Node test runner, Vite
